# 作文的准备工作

## 原则

```
在战斗中寻找战斗的方法
要考虑文章是否容易被理解
以内容和逻辑取胜
```


## 里程碑

- [x] 明确研究项目的目的或目标
- [ ] 有要求的话，写出项目假设
- [ ] 选择一个准确、清楚的题目
- [ ] 查找文献，写出文献综述
- [ ] 决定用什么研究方法，并写出来
- [ ] 问问导师这个项目的主题和范围是否合适
- [ ] 参与讨论
- [ ] 找到准确的搜集和记录数据的格式
- [ ] 并整理数据
- [ ] 分析数据得出结果和讨论
- [ ] 完成报告
- [ ] 所有必须写的部分是不是都写了，完成了？
- [ ] 每个部分都有清楚的标题吗
- [ ] 每个部分有没有准确的材料
- [ ] 报告符合字数要求吗
- [ ] 读者有没有获得他们需要的信息，所有信息都相关吗
- [ ] 写出摘要，放到标题页之后
- [ ] 标记页码
- [ ] 目录页
- [ ] 检查写作风格
- [ ] 检查通篇是否清楚易读
- [ ] 报告是否校对
